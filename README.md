# 👨‍👩‍👧‍👦 Family RP Bot

Telegram-бот для ролевой игры в семью в групповых чатах.

## ✨ Возможности

- 💍 **Браки** — предлагай брак друзьям
- 💔 **Разводы** — расставайся с подтверждением
- 👶 **Усыновления** — стань родителем других участников
- 👨‍👩‍👧‍👦 **Семейное древо** — смотри свою семью
- 🎭 **Роли** — Лупа, Пупа, Залупа, Залупомэн
- 👤 **Профили** — статистика и информация

## 🚀 Установка

1. Клонируй репозиторий
2. Установи зависимости: `pip install -r requirements.txt`
3. Укажи токен в `config.py`
4. Запусти: `python main.py`

## 📦 Технологии

- Python 3.9+
- aiogram 3.4.1
- aiosqlite 0.19.0
